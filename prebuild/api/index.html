LSS-Manager API

Usage: GET https://api.lss-manager.de/{lang}/{type}

The following values are allowed for {lang}:

  • cs_CZ
  • de_DE
  • en_AU
  • en_GB
  • en_US
  • es_ES
  • fr_FR
  • it_IT
  • nb_NO
  • nl_NL
  • pl_PL
  • sv_SE

The following values are allowed for {type}:

  • vehicles
    • Types: interface BackwardsCompatibilityVehicle
  • vehicleCategories
    • Types: interface VehicleCategory
  • buildings
    • Types: interface InternalBuilding
  • buildingCategories
    • Types: interface BuildingCategory
  • schoolings
    • Types: interface SchoolingsBySchool
  • ranks
    • Type: { missionchief: Record<number, string>; policechief: Record<number, string>; }
  • pois
    • Type: string[]
  • small_buildings
    • Type: Record<number, number>
  • einsaetze
    • giving translations to the keys in the /einsaetze.json file of each language
Note, that not all languages may provide values for all types. Values returned are always similar to those in the latest LSS-Manager V.4 beta build
